# ⚽ Sistema Torneos Futbol

Sistema web para la administración de torneos de fútbol desarrollado por un equipo de estudiantes con el objetivo de aprender desarrollo web utilizando Flask.

## 🚀 Tecnologías

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- Git & GitHub

## 📌 Funcionalidades

- Inicio de sesión
- Registro de usuarios
- Gestión de equipos
- Gestión de jugadores
- Gestión de torneos
- Programación de partidos
- Registro de resultados
- Tabla de posiciones
- Estadísticas
